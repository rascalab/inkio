# 어댑터

Inkio는 코어 어댑터와 확장 어댑터를 분리합니다.

## Core

```ts
import type { InkioAdapter } from '@inkio/editor';

const adapter: InkioAdapter = {
  onCreate: (editor) => {
    console.log('ready', editor);
  },
  onUpdate: (content) => {
    console.log(content);
  },
  onError: (error, ctx) => {
    console.error(ctx.source, error);
  },
};
```

## Advanced

```ts
import type { ExtensionsAdapter } from '@inkio/advanced';

const adapter: ExtensionsAdapter = {
  suggestion: {
    mentionItems: async (query) => searchUsers(query),
    hashtagItems: async (query) => searchTags(query),
    slashCommands: async (query) => searchCommands(query),
  },
  navigation: {
    onWikiLinkClick: (href) => router.push(`/wiki/${href}`),
  },
};
```

## 적용

`@inkio/editor`와 `@inkio/simple`의 high-level wrapper에서는 보통 `defaultExtensionsOptions`를 직접 쓰는 편이 더 명확합니다.
