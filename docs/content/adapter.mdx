# 어댑터 (Deprecated)

> v0.0.5부터 어댑터 패턴은 제거되었습니다. `@inkio/editor`와 `@inkio/simple`은 flat props API를 사용합니다.

## 이전 방식 → 현재 방식

이전에는 `ExtensionsAdapter` 객체를 구성해서 전달했지만, 이제는 Editor 컴포넌트에 직접 props로 전달합니다:

```tsx
// Before (removed)
const adapter = {
  suggestion: { hashtagItems, mentionItems },
  navigation: { onWikiLinkClick },
};
<Editor adapter={adapter} />

// After (current)
<Editor
  hashtagItems={hashtagItems}
  mentionItems={mentionItems}
  onWikiLinkClick={onWikiLinkClick}
/>
```

자세한 API는 [컴포넌트](/components/) 문서를 참고하세요.
