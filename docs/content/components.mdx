# 컴포넌트

## 주요 표면

`@inkio/editor`와 `@inkio/simple`은 둘 다 아래 core UI surface를 다시 export합니다.

- `Editor`
- `Viewer`
- `Toolbar`
- `BubbleMenu`
- `FloatingMenu`
- `TableMenu`

## Viewer 목차

`Viewer`는 built-in ToC를 지원합니다.

```tsx
import { Viewer } from '@inkio/editor';

<Viewer
  content={content}
  tableOfContents={{ position: 'right', maxLevel: 3 }}
/>;
```

외부 레이아웃에 별도 ToC를 렌더하고 싶으면 `onHeadingsReady`로 heading 목록과 `scrollToIndex` helper를 받을 수 있습니다.

## 공통 UI override

공통 prop:

- `locale`
- `messages`
- `icons`

```tsx
import { Editor, type InkioMessageOverrides } from '@inkio/editor';
import { inkioIconRegistry } from '@inkio/editor/icons';

const messages: InkioMessageOverrides = {
  core: {
    suggestion: { empty: 'No matching items.' },
  },
};

const icons = {
  comment: inkioIconRegistry.comment,
};

export function Page() {
  return <Editor messages={messages} icons={icons} />;
}
```
