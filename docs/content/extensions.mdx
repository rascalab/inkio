# 확장

`@inkio/extension`는 선택형 확장 모음입니다.

- 루트 import: `@inkio/extension`
- 서브패스 import: `@inkio/extension/mention` 등

## 빠른 구성

```tsx
import { Editor } from '@inkio/editor';
import { getDefaultInkioExtensions } from '@inkio/extension';
import '@inkio/editor/minimal.css';
import '@inkio/extension/style.css';

const extensions = getDefaultInkioExtensions({
  placeholder: 'Start typing...',
  blockHandle: true,
  locale: 'en-US,en;q=0.9',
  messages: {
    extensions: {
      blockHandle: {
        transformSection: 'Convert to',
      },
    },
  },
});

export function Page() {
  return <Editor initialContent="" extensions={extensions} />;
}
```

## BlockHandle 기본 메뉴

`blockHandle: true`를 켜면 드래그 핸들과 기본 액션 메뉴가 함께 활성화됩니다.

기본 메뉴는 다음을 포함합니다.

- 삭제 / 복제
- Paragraph / Heading 변환
- Bullet / Ordered list 변환
- Quote / Callout / Code block 변환

동작 특성:

- 뷰포트 경계 충돌 시 flip/shift 클램프 적용
- 스크롤/리사이즈 중 위치 자동 업데이트
- `Escape`, `ArrowUp/Down`, `Enter`, `Space` 키보드 탐색 지원
- 기존 커스텀 이벤트(`inkio:block-menu`)도 계속 발행

### BlockHandle 커스터마이징

```tsx
import { BlockHandle } from '@inkio/editor';

const customBlockHandle = BlockHandle.configure({
  locale: 'en-US',
  messages: {
    extensions: {
      blockHandle: {
        duplicate: 'Clone',
      },
    },
  },
  icons: {
    duplicate: <span>+</span>,
  },
});
```

> `icons`는 `BlockMenuIcons` 형태입니다.

## Comment UI

`CommentPanel`, `CommentComposer`는 `locale/messages/icons`를 직접 받습니다.

```tsx
<CommentPanel
  editor={editor}
  threads={threads}
  locale="en-US"
  messages={{
    extensions: {
      commentPanel: {
        resolve: 'Done',
      },
    },
  }}
  onReply={onReply}
  onResolve={onResolve}
  onDelete={onDelete}
/>
```

## ImageEditor

`ImageEditorModal`은 `locale/messages/icons`를 지원합니다.

```tsx
<ImageEditorModal
  isOpen={open}
  imageSrc={src}
  locale="en-US,en;q=0.8"
  messages={{
    extensions: {
      imageEditor: {
        save: 'Apply',
      },
    },
  }}
  onSave={handleSave}
  onClose={handleClose}
/>
```
