# 확장

Inkio 확장은 두 층으로 나뉩니다.

- `@inkio/essential`
  - callout
  - details/toggle
  - table
  - keyboard shortcuts
- `@inkio/advanced`
  - essential 전부 포함
  - slash command
  - block handle
  - mention
  - hashtag
  - wiki link
  - bookmark
  - comment

## 권장 시작점

- classic editor가 필요하면 `@inkio/simple`
- notion-like editor가 필요하면 `@inkio/editor`
- low-level 조합이 필요하면 `@inkio/core` + `@inkio/essential` 또는 `@inkio/advanced`

## 조합 예시

```tsx
import { Editor } from '@inkio/core';
import { getDefaultExtensions } from '@inkio/advanced';
import '@inkio/core/minimal.css';
import '@inkio/advanced/style.css';

const extensions = getDefaultExtensions({
  hashtagItems: ({ query }) => [{ id: query || 'inkio', label: `#${query || 'inkio'}` }],
});

export function Page() {
  return <Editor extensions={extensions} initialContent="<p>Hello Inkio</p>" />;
}
```

## Image editor

```tsx
import { Editor } from '@inkio/editor';
import { ImageEditorModal } from '@inkio/image-editor';
import '@inkio/image-editor/style.css';

export function Page() {
  return (
    <Editor
      defaultExtensionsOptions={{
        imageBlock: {
          onUpload: async (file) => URL.createObjectURL(file),
          imageEditor: ImageEditorModal,
        },
      }}
    />
  );
}
```

## CSS

- `@inkio/simple/minimal.css`
- `@inkio/editor/minimal.css`
- `@inkio/advanced/style.css` for direct `@inkio/core` + `@inkio/advanced` composition
- `@inkio/image-editor/style.css`
