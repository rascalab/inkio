# i18n & Locale

Inkio는 `locale`과 `messages` prop으로 UI 문구를 제어합니다.

## locale 입력

- 문자열: `en`, `ko-KR`
- Accept-Language 문자열
- 배열
- `Intl.Locale`
- locale-like object

## 메시지 override

```tsx
import type { InkioMessageOverrides } from '@inkio/editor';

const messages: InkioMessageOverrides = {
  core: {
    actions: {
      comment: 'Discuss',
    },
  },
  extensions: {
    commentComposer: {
      placeholder: 'Leave a note...',
    },
    imageEditor: {
      save: 'Apply',
    },
  },
};
```

## 기본 메시지셋

```tsx
import { enCoreMessages } from '@inkio/editor';
import { enCommentMessages } from '@inkio/advanced';
import { enImageEditorMessages } from '@inkio/image-editor';
```
