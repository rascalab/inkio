# Inkio

Inkio는 Tiptap 기반의 레이어드 에디터 패키지 모음입니다.

## 시작점

- `@inkio/simple`
  - classic WYSIWYG
  - toolbar-first
  - 가장 쉬운 시작점
- `@inkio/editor`
  - notion-like
  - bubble/floating menu, slash command, block handle 중심
- `@inkio/advanced`
  - comments, mention, hashtag, wiki link 같은 고급 확장
- `@inkio/image-editor`
  - `ImageBlock`용 이미지 편집 UI

## 빠른 설치

Classic:

```bash
npm install @inkio/simple react react-dom
```

Notion-like:

```bash
npm install @inkio/editor react react-dom
```

옵션 패키지:

```bash
npm install @inkio/image-editor
```

`CommentPanel` 같은 advanced export를 앱 코드에서 직접 import할 때만 `@inkio/advanced`를 추가 설치합니다.

## 예제

```tsx
import { Editor } from '@inkio/simple';
import '@inkio/simple/minimal.css';

export function Page() {
  return <Editor initialContent="<p>Hello Inkio</p>" />;
}
```

Markdown은 서브패스로 제공합니다.

```tsx
import { parseMarkdown, stringifyMarkdown } from '@inkio/simple/markdown';
```

구현은 `remark/unified` 기반이며 markdown과 `JSONContent`를 직접 변환합니다.
