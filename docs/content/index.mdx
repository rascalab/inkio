# Inkio

React 기반의 커스터마이즈 가능한 리치 텍스트 에디터 라이브러리입니다.

## 주요 기능

- 🎨 **Headless 디자인** - 완전한 스타일 커스터마이징
- 📝 **풍부한 포맷팅** - Bold, Italic, Underline, Highlight, Link 등
- 🖼️ **고급 이미지 편집** - 드래그앤드롭, 리사이즈, Crop, 마크업
- 💬 **Callout 블록** - Info, Warning, Tip, Danger 타입 지원
- 🔗 **위키 링크** - `[[페이지]]` 형식의 내부 링크
- 👤 **멘션 시스템** - `@사용자` 태그 지원
- #️⃣ **해시태그** - `#태그` 지원
- ⌨️ **슬래시 커맨드** - `/` 입력으로 빠른 블록 삽입
- 📋 **태스크 리스트** - 체크박스 기반 할일 목록
- 🎯 **Tree-shakable** - 필요한 기능만 번들링

## 빠른 시작

```bash
npm install @inkio/editor @inkio/extension react react-dom
```

`@inkio/extension`는 `@inkio/editor`, `react`, `react-dom`만 peer로 요구합니다.
Tiptap 런타임과 `katex`, `konva`, `react-konva`는 `@inkio/extension` 내부 의존성으로 제공됩니다.

```tsx
import { Editor, getDefaultCoreExtensions } from '@inkio/editor';
import '@inkio/editor/style.css';

function MyEditor() {
  return (
    <Editor
      extensions={getDefaultCoreExtensions()}
      onUpdate={(json) => console.log(json)}
    />
  );
}
```

## 기술 스택

- [Tiptap](https://tiptap.dev) v3 - ProseMirror 기반 에디터 프레임워크
- [React](https://react.dev) 19+ - UI 라이브러리
- [TypeScript](https://www.typescriptlang.org) - 타입 안전성
- [Tailwind CSS](https://tailwindcss.com) v4 - 스타일링
- [Konva](https://konvajs.org) + [react-konva](https://konvajs.org/docs/react/) - 이미지 편집 (Crop, Rotate, Draw, Shapes, Text)
