# Basic React Recipe

`examples/basic-react`는 `@inkio/simple` 기준 예제입니다.

적합한 경우:

- classic WYSIWYG 편집기
- toolbar-first UI
- 가장 작은 React 시작점

실행:

```bash
pnpm install
pnpm --filter example-basic-react dev
```

핵심 포인트:

- `@inkio/simple/minimal.css`
- `Editor` wrapper 기본 preset 사용
- 추가 확장이 필요하면 `extensions` prop으로 직접 조합
