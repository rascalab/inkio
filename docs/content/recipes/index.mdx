# Integration Recipes

Inkio를 AI와 함께 빠르게 붙일 때는 설명형 문서보다 `가장 가까운 실행 가능한 예제`를 기준으로 시작하는 편이 더 안정적입니다.

추천 순서:

1. 가장 가까운 예제를 고릅니다.
2. 루트의 `AI_CONTEXT.md`를 같이 봅니다.
3. 필요한 기능만 예제 위에 덧붙입니다.

바로 쓸 수 있는 기준 예제:

- [Basic React](/recipes/basic-react) - Vite + React, `@inkio/editor`만 사용
- [Next App Router](/recipes/next-app-router) - Next.js App Router, `@inkio/editor` + `@inkio/extension`

레포 안 실제 경로:

- `examples/basic-react`
- `examples/next-app-router`
- `AI_CONTEXT.md`
