# Next App Router Recipe

`examples/next-app-router`는 `@inkio/editor` 기준 예제입니다.

적합한 경우:

- notion-like editor
- Next.js App Router
- comments와 image editor를 함께 쓰는 경우

실행:

```bash
pnpm install
pnpm --filter example-next-app-router dev
```

핵심 포인트:

- editor는 `'use client';` 컴포넌트 안에서 렌더링
- `@inkio/editor/minimal.css`, `@inkio/image-editor/style.css` import
- `transpilePackages`에 `@inkio/editor`, `@inkio/advanced`, `@inkio/image-editor` 추가
