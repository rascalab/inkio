# 직렬화

기본 직렬화 유틸리티는 `@inkio/editor`와 `@inkio/simple`에서 다시 export합니다.

```tsx
import { toPlainText, toSummary, getContentStats } from '@inkio/editor';
import { extractMentions, extractHashtags } from '@inkio/advanced';
```

Markdown import/export는 별도 subpath로 제공합니다.

```tsx
import { parseMarkdown, stringifyMarkdown } from '@inkio/editor/markdown';
```

Markdown round-trip은 `core + essential` 범위까지만 보장합니다.
구현은 `remark/unified` 기반이며 HTML bridge를 사용하지 않습니다.
